One bot, several uses.

Moderation

All the moderation commands you'll need! The mute command can't even be bypassed by re-joining the server.
1Bot also has a command group to manage roles and possibly the best snipe command ever.

Utilities

1Bot has lots of utility commands, some of which are not in most other bots. Like the weather command!

Fun

Most multipurpose bots lack fun commands.
1Bot has all you need: meme, dog, cat, youtube, chess, 8ball & more.

Images

Some fun image generation commands like Triggered, Amogus, Tweet etc.
